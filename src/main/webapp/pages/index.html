API Design

(GET) /api/v1/jewelry -- list all jewelry data

(GET) /api/v1/jewelry/{id} -- get single jewelry data

(POST) /api/v1/jewelry -- save jewelry

(DELETE) /api/v1/jewelry/{id} -- deletes single jewelry

Normal users have ROLE_USER role. Admins have ROLE_ADMIN role.

Add Jewelry Page is only for ROLE_ADMIN users

Server side validations added to user register form. JSR 303
